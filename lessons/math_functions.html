الوحدة 1-2

القاعدة 1: المجال والمدى (Domain & Range)

المجال: كل قيم \\(x\\) المسموح بها. المدى: كل قيم \\(y\\) الناتجة.

القواعد الأساسية لإيجاد المجال:
المقام: \\( \\neq 0 \\)
الجذر الزوجي: ما بداخله \\( \\geq 0 \\)
اللوغاريتم: الحجة \\( > 0 \\)
اختبار الخط الرأسي: أي خط رأسي يقطع المنحنى مرة واحدة فقط.
Module 1-2

Rule 1: Domain & Range

Domain: All allowed \\(x\\)-inputs. Range: All possible \\(y\\)-outputs.

Basic Rules for Domain:
Denominator: \\( \\neq 0 \\)
Even Root: Inner part \\( \\geq 0 \\)
Logarithm: Argument \\( > 0 \\)
Vertical Line Test: Any vertical line cuts the curve only once.

📌 مثال عملي (Practical Example):

أوجد مجال الدالة \\( f(x) = \\frac{1}{x-3} + \\sqrt{x-1} \\).

الحل التكتيكي:
1. للكسر: المقام \\( x-3 \\neq 0 \\implies x \\neq 3 \\).
2. للجذر: \\( x-1 \\geq 0 \\implies x \\geq 1 \\).
3. الدمج: كل الأرقام من 1 إلى ∞ باستثناء 3.
النتيجة: \\( [1, 3) \\cup (3, \\infty) \\).

📌 Illustrative Example:

Find the domain of \\( f(x) = \\frac{1}{x-3} + \\sqrt{x-1} \\).

Tactical Solution:
1. For fraction: \\( x-3 \\neq 0 \\implies x \\neq 3 \\).
2. For root: \\( x-1 \\geq 0 \\implies x \\geq 1 \\).
3. Merge: from 1 to ∞ except 3.
Result: \\( [1, 3) \\cup (3, \\infty) \\).

الوحدة 3

القاعدة 2: رتابة الدوال (Monotonicity)

الدالة \\(f(x)\\) تسمى متزايدة إذا كانت \\(x_1 < x_2 \\implies f(x_1) < f(x_2)\\).
وتسمى متناقصة إذا كانت \\(x_1 < x_2 \\implies f(x_1) > f(x_2)\\).

لتحديد الرتابة باستخدام المشتقة:
\\( f'(x) > 0 \\) ⟹ الدالة متزايدة في تلك الفترة
\\( f'(x) < 0 \\) ⟹ الدالة متناقصة في تلك الفترة
\\( f'(x) = 0 \\) ⟹ نقطة حرجة (قيم قصوى أو انعطاف)
Module 3

Rule 2: Monotonicity

\\(f(x)\\) is increasing if \\(x_1 < x_2 \\implies f(x_1) < f(x_2)\\).
Decreasing if \\(x_1 < x_2 \\implies f(x_1) > f(x_2)\\).

Using derivative:
\\( f'(x) > 0 \\) ⟹ increasing in that interval
\\( f'(x) < 0 \\) ⟹ decreasing in that interval
\\( f'(x) = 0 \\) ⟹ critical point (extremum or inflection)

📌 مثال تطبيقي (Application):

حدد فترات التزايد والتناقص للدالة \\( f(x) = x^2 - 4x + 3 \\).

الحل:
1. المشتقة: \\( f'(x) = 2x - 4 \\).
2. النقطة الحرجة: \\( 2x - 4 = 0 \\implies x = 2 \\).
3. نختبر القيم: \\( x=0 \\) (قبل 2) ⟹ \\( f'(0) = -4 < 0 \\) ⟹ متناقصة.
4. \\( x=3 \\) (بعد 2) ⟹ \\( f'(3) = 2 > 0 \\) ⟹ متزايدة.
النتيجة: متناقصة على \\((-\\infty, 2)\\)، متزايدة على \\((2, \\infty)\\).

📌 Application Example:

Find the intervals of increase/decrease for \\( f(x) = x^2 - 4x + 3 \\).

Solution:
1. Derivative: \\( f'(x) = 2x - 4 \\).
2. Critical point: \\( x = 2 \\).
3. Test \\(x=0\\): \\( f'(0) = -4 < 0 \\) ⟹ decreasing.
4. Test \\(x=3\\): \\( f'(3) = 2 > 0 \\) ⟹ increasing.
Result: Decreasing on \\((-\\infty, 2)\\), increasing on \\((2, \\infty)\\).

الوحدة 4

القاعدة 3: الدوال العكسية (Inverse Functions)

الدالة العكسية \\(f^{-1}(x)\\) تعكس تأثير \\(f(x)\\). شرط الوجود: الدالة يجب أن تكون تقابلية (One-to-One) — أي تجتاز اختبار الخط الأفقي.

خطوات إيجاد الدالة العكسية:
1. اكتب \\( y = f(x) \\).
2. بدل \\(x\\) و \\(y\\).
3. حل المعادلة الجديدة لـ \\(y\\).
4. الناتج هو \\( f^{-1}(x) \\).

بيانياً: \\(f^{-1}\\) هي انعكاس \\(f\\) حول الخط \\(y = x\\).
Module 4

Rule 3: Inverse Functions

The inverse function \\(f^{-1}(x)\\) reverses \\(f(x)\\). Existence condition: \\(f\\) must be One-to-One (passes the Horizontal Line Test).

Steps to find the inverse:
1. Write \\( y = f(x) \\).
2. Swap \\(x\\) and \\(y\\).
3. Solve the new equation for \\(y\\).
4. The result is \\( f^{-1}(x) \\).

Graphically: \\(f^{-1}\\) is the reflection of \\(f\\) about the line \\(y = x\\).

📌 مثال تطبيقي (Application):

أجد الدالة العكسية لـ \\( f(x) = 2x + 3 \\).

الحل:
1. \\( y = 2x + 3 \\).
2. نبدل: \\( x = 2y + 3 \\).
3. نحل: \\( 2y = x - 3 \\implies y = \\frac{x-3}{2} \\).
النتيجة: \\( f^{-1}(x) = \\frac{x-3}{2} \\).
تحقق: \\( f(f^{-1}(x)) = 2(\\frac{x-3}{2}) + 3 = x-3+3 = x \\). ✓

📌 Application Example:

Find the inverse of \\( f(x) = 2x + 3 \\).

Solution:
1. \\( y = 2x + 3 \\).
2. Swap: \\( x = 2y + 3 \\).
3. Solve: \\( 2y = x - 3 \\implies y = \\frac{x-3}{2} \\).
Result: \\( f^{-1}(x) = \\frac{x-3}{2} \\).
Check: \\( f(f^{-1}(x)) = 2(\\frac{x-3}{2}) + 3 = x \\). ✓

الوحدة 5

القاعدة 4: الدوال الأسية واللوغاريتمية

الدالة الأسية: \\( f(x) = a^x \\) حيث \\(a > 0, a \\neq 1\\).
مجالها \\(\\mathbb{R}\\) ومداها \\((0, \\infty)\\).

الدالة اللوغاريتمية: \\( f(x) = \\log_a(x) \\) هي عكس الأسية.
مجالها \\((0, \\infty)\\) ومداها \\(\\mathbb{R}\\).

قوانين اللوغاريتمات:
\\( \\log_a(MN) = \\log_a M + \\log_a N \\)
\\( \\log_a(M/N) = \\log_a M - \\log_a N \\)
\\( \\log_a(M^n) = n \\log_a M \\)
\\( \\log_a(a) = 1, \\quad \\log_a(1) = 0 \\)
\\( \\log_a x = \\frac{\\ln x}{\\ln a} \\) (تغيير الأساس)
Module 5

Rule 4: Exponential & Logarithmic Functions

Exponential: \\( f(x) = a^x \\) where \\(a > 0, a \\neq 1\\).
Domain \\(\\mathbb{R}\\), Range \\((0, \\infty)\\).

Logarithmic: \\( f(x) = \\log_a(x) \\) is inverse of exponential.
Domain \\((0, \\infty)\\), Range \\(\\mathbb{R}\\).

Logarithm Rules:
\\( \\log_a(MN) = \\log_a M + \\log_a N \\)
\\( \\log_a(M/N) = \\log_a M - \\log_a N \\)
\\( \\log_a(M^n) = n \\log_a M \\)
\\( \\log_a(a) = 1, \\quad \\log_a(1) = 0 \\)
\\( \\log_a x = \\frac{\\ln x}{\\ln a} \\) (Change of Base)

📌 مثال تطبيقي (Application):

حل المعادلة: \\( \\log_2(x) + \\log_2(4) = 5 \\).

الحل:
1. ندمج اللوغاريتمات: \\( \\log_2(4x) = 5 \\).
2. نحول للصيغة الأسية: \\( 4x = 2^5 \\).
3. الحساب: \\( 4x = 32 \\implies x = 8 \\).
التحقق: \\( \\log_2(8) + \\log_2(4) = 3 + 2 = 5 \\) ✓

📌 Application Example:

Solve: \\( \\log_2(x) + \\log_2(4) = 5 \\).

Solution:
1. Merge Logs: \\( \\log_2(4x) = 5 \\).
2. Exponential form: \\( 4x = 2^5 \\).
3. Calculate: \\( 4x = 32 \\implies x = 8 \\).
Check: \\( \\log_2(8) + \\log_2(4) = 3 + 2 = 5 \\) ✓

الوحدة 6

القاعدة 5: الدوال المثلثية (Trigonometric Functions)

الدوال المثلثية الأساسية الست: \\(\\sin x, \\cos x, \\tan x, \\cot x, \\sec x, \\csc x\\).

قيم شهيرة:
\\( \\sin 0 = 0, \\sin \\frac{\\pi}{2} = 1, \\sin \\pi = 0 \\)
\\( \\cos 0 = 1, \\cos \\frac{\\pi}{2} = 0, \\cos \\pi = -1 \\)
\\( \\tan x = \\frac{\\sin x}{\\cos x}, \\quad \\cot x = \\frac{\\cos x}{\\sin x} \\)

الدورة: \\(\\sin, \\cos, \\sec, \\csc\\) دورتها \\(2\\pi\\)، \\(\\tan, \\cot\\) دورتها \\(\\pi\\).
Module 6

Rule 5: Trigonometric Functions

The six basic trig functions: \\(\\sin x, \\cos x, \\tan x, \\cot x, \\sec x, \\csc x\\).

Notable Values:
\\( \\sin 0 = 0, \\sin \\frac{\\pi}{2} = 1, \\sin \\pi = 0 \\)
\\( \\cos 0 = 1, \\cos \\frac{\\pi}{2} = 0, \\cos \\pi = -1 \\)
\\( \\tan x = \\frac{\\sin x}{\\cos x}, \\quad \\cot x = \\frac{\\cos x}{\\sin x} \\)

Period: \\(\\sin, \\cos, \\sec, \\csc\\) have period \\(2\\pi\\), \\(\\tan, \\cot\\) have period \\(\\pi\\).

📌 مثال تطبيقي (Application):

أوجد قيمة \\( \\sin(\\frac{\\pi}{6}) + \\cos(\\frac{\\pi}{3}) \\).

الحل:
1. \\( \\sin(\\frac{\\pi}{6}) = \\frac{1}{2} \\).
2. \\( \\cos(\\frac{\\pi}{3}) = \\frac{1}{2} \\).
3. المجموع: \\( \\frac{1}{2} + \\frac{1}{2} = 1 \\).
النتيجة: 1.

📌 Application Example:

Find \\( \\sin(\\frac{\\pi}{6}) + \\cos(\\frac{\\pi}{3}) \\).

Solution:
1. \\( \\sin(\\frac{\\pi}{6}) = \\frac{1}{2} \\).
2. \\( \\cos(\\frac{\\pi}{3}) = \\frac{1}{2} \\).
3. Sum: \\( \\frac{1}{2} + \\frac{1}{2} = 1 \\).
Result: 1.

الوحدة 7

القاعدة 6: النهايات (Limits)

النهاية تصف سلوك الدالة عندما تقترب \\(x\\) من قيمة معينة. الرمز: \\( \\lim_{x \\to a} f(x) = L \\).

قواعد أساسية:
\\( \\lim_{x \\to a} c = c \\) (نهاية الثابت)
\\( \\lim_{x \\to a} x^n = a^n \\)
\\( \\lim_{x \\to a} [f(x) \\pm g(x)] = \\lim f(x) \\pm \\lim g(x) \\)
\\( \\lim_{x \\to a} [f(x) \\cdot g(x)] = \\lim f(x) \\cdot \\lim g(x) \\)

نهايات شهيرة:
\\( \\lim_{x \\to 0} \\frac{\\sin x}{x} = 1 \\)
\\( \\lim_{x \\to 0} \\frac{e^x - 1}{x} = 1 \\)
Module 7

Rule 6: Limits

A limit describes function behavior as \\(x\\) approaches a value. Notation: \\( \\lim_{x \\to a} f(x) = L \\).

Basic Rules:
\\( \\lim_{x \\to a} c = c \\) (constant limit)
\\( \\lim_{x \\to a} x^n = a^n \\)
\\( \\lim_{x \\to a} [f(x) \\pm g(x)] = \\lim f(x) \\pm \\lim g(x) \\)
\\( \\lim_{x \\to a} [f(x) \\cdot g(x)] = \\lim f(x) \\cdot \\lim g(x) \\)

Famous Limits:
\\( \\lim_{x \\to 0} \\frac{\\sin x}{x} = 1 \\)
\\( \\lim_{x \\to 0} \\frac{e^x - 1}{x} = 1 \\)

📌 مثال تطبيقي (Application):

احسب \\( \\lim_{x \\to 2} \\frac{x^2 - 4}{x - 2} \\).

الحل:
1. عند التعويض المباشر: \\( \\frac{4-4}{2-2} = \\frac{0}{0} \\) (غير معين).
2. نحلل البسط: \\( \\frac{(x-2)(x+2)}{x-2} \\).
3. نختصر: \\( \\lim_{x \\to 2} (x+2) \\).
4. نعوض: \\( 2 + 2 = 4 \\).
النتيجة: 4.

📌 Application Example:

Calculate \\( \\lim_{x \\to 2} \\frac{x^2 - 4}{x - 2} \\).

Solution:
1. Direct substitution: \\( \\frac{0}{0} \\) (indeterminate).
2. Factor numerator: \\( \\frac{(x-2)(x+2)}{x-2} \\).
3. Cancel: \\( \\lim_{x \\to 2} (x+2) \\).
4. Substitute: \\( 2 + 2 = 4 \\).
Result: 4.

الوحدة 8

القاعدة 7: المشتقات (Derivatives)

المشتقة تقيس معدل التغير اللحظي. المشتقة الأولى \\(f'(x)\\) هي ميل المماس عند أي نقطة.

قواعد الاشتقاق الأساسية:
قاعدة القوة: \\( \\frac{d}{dx}(x^n) = nx^{n-1} \\)
قاعدة الثابت: \\( \\frac{d}{dx}(c) = 0 \\)
قاعدة الجداء: \\( (fg)' = f'g + fg' \\)
قاعدة السلسلة: \\( (f(g(x)))' = f'(g(x)) \\cdot g'(x) \\)
مشتقة الأس: \\( \\frac{d}{dx}(e^x) = e^x \\)
مشتقة اللوغاريتم: \\( \\frac{d}{dx}(\\ln x) = \\frac{1}{x} \\)
مشتقة المثلثات: \\( \\frac{d}{dx}(\\sin x) = \\cos x, \\frac{d}{dx}(\\cos x) = -\\sin x \\)
Module 8

Rule 7: Derivatives

The derivative measures the instantaneous rate of change. \\(f'(x)\\) is the slope of the tangent line at any point.

Basic Differentiation Rules:
Power Rule: \\( \\frac{d}{dx}(x^n) = nx^{n-1} \\)
Constant Rule: \\( \\frac{d}{dx}(c) = 0 \\)
Product Rule: \\( (fg)' = f'g + fg' \\)
Chain Rule: \\( (f(g(x)))' = f'(g(x)) \\cdot g'(x) \\)
Exponential: \\( \\frac{d}{dx}(e^x) = e^x \\)
Logarithm: \\( \\frac{d}{dx}(\\ln x) = \\frac{1}{x} \\)
Trig: \\( \\frac{d}{dx}(\\sin x) = \\cos x, \\frac{d}{dx}(\\cos x) = -\\sin x \\)

📌 مثال تطبيقي (Application):

أوجد مشتقة \\( f(x) = (2x+1)^3 \\) باستخدام قاعدة السلسلة.

الحل:
1. الدالة الخارجية: \\( u^3 \\)، مشتقتها \\( 3u^2 \\).
2. الدالة الداخلية: \\( u=2x+1 \\)، مشتقتها \\( 2 \\).
3. قاعدة السلسلة: \\( f'(x) = 3(2x+1)^2 \\cdot 2 \\).
النتيجة: \\( f'(x) = 6(2x+1)^2 \\).
تحقق: نفك القوس: \\( (2x+1)^3 = 8x^3+12x^2+6x+1 \\)، مشتقتها \\( 24x^2+24x+6 = 6(4x^2+4x+1) = 6(2x+1)^2 \\). ✓

📌 Application Example:

Find derivative of \\( f(x) = (2x+1)^3 \\) using the Chain Rule.

Solution:
1. Outer: \\( u^3 \\), derivative \\( 3u^2 \\).
2. Inner: \\( u=2x+1 \\), derivative \\( 2 \\).
3. Chain Rule: \\( f'(x) = 3(2x+1)^2 \\cdot 2 \\).
Result: \\( f'(x) = 6(2x+1)^2 \\).
Check: Expand \\( (2x+1)^3 \\), derivative is \\( 24x^2+24x+6 = 6(2x+1)^2 \\). ✓

🎯 نصيحة CSCA

  • تذكر أن \\( \\log(0) \\) و \\( \\log(\\text{سالب}) \\) غير معرفين تماماً — يأتي هذا في أسئلة المجال.
  • الدوال العكسية أكثرها طلباً في الاختبار هي \\(e^x\\) مع \\(\\ln x\\)، والدوال الخطية البسيطة.
  • لا تنس قاعدة السلسلة! أكثر من نصف أسئلة المشتقات في CSCA تحتاجها.
  • في النهايات، إذا واجهت \\(\\frac{0}{0}\\)، ابحث عن تحليل أو مرافق للجذور.
  • لحفظ الدورة المثلثية: "All Students Take Calculus" — كل موجب في الربع الأول.
  • في الرتابة، استخدم المشتقة الأولى دائماً ولا تنس فترات الاختبار.

🎯 CSCA Exam Tips

  • Remember \\( \\log(0) \\) and \\( \\log(\\text{negative}) \\) are undefined — they appear in domain questions.
  • Most commonly tested inverse functions: \\(e^x\\) with \\(\\ln x\\), and simple linear functions.
  • Don't forget the Chain Rule! Over half of derivative questions in CSCA require it.
  • For limits with \\(\\frac{0}{0}\\), try factoring or rationalizing the numerator.
  • Remember trig periods: "All Students Take Calculus" — positivity in each quadrant.
  • For monotonicity, always use the first derivative and don't forget test intervals.
