الوحدة 1

القاعدة 1: المستقيمات (Lines)

ميل المستقيم (\\(m\\)) يمثل "انحدار" الخط. معادلة المستقيم تُكتب بعدة صيغ.

صيغ معادلة المستقيم:
صيغة الميل والنقطة: \\( y - y_1 = m(x - x_1) \\)
صيغة الميل والمقطع: \\( y = mx + b \\)
الصيغة العامة: \\( Ax + By + C = 0 \\)

التوازي: \\( m_1 = m_2 \\)
التعامد: \\( m_1 \\cdot m_2 = -1 \\)
Module 1

Rule 1: Lines

Slope (\\(m\\)) represents line steepness. Line equations come in several forms.

Line Equation Forms:
Point-Slope: \\( y - y_1 = m(x - x_1) \\)
Slope-Intercept: \\( y = mx + b \\)
General Form: \\( Ax + By + C = 0 \\)

Parallel: \\( m_1 = m_2 \\)
Perpendicular: \\( m_1 \\cdot m_2 = -1 \\)

📌 مثال تطبيقي (Application):

أوجد ميل المستقيم العمودي على المستقيم المار بالنقاط \\( (2, 3) \\) و \\( (4, 7) \\).

الحل:
1. ميل المستقيم الأصلي: \\( m = \\frac{7-3}{4-2} = \\frac{4}{2} = 2 \\).
2. ميل العمودي: \\( m_1 \\cdot m_2 = -1 \\implies 2 \\cdot m_2 = -1 \\).
النتيجة: \\( m_2 = -\\frac{1}{2} \\).
3. معادلة العمودي المار بنقطة (2,3): \\( y-3 = -\\frac{1}{2}(x-2) \\).

📌 Application Example:

Find slope of a line perpendicular to the line through \\( (2, 3) \\) and \\( (4, 7) \\).

Solution:
1. Original slope: \\( m = \\frac{7-3}{4-2} = 2 \\).
2. Perpendicular slope: \\( m_2 = -\\frac{1}{2} \\).
Result: \\( -\\frac{1}{2} \\).
3. Equation through (2,3): \\( y-3 = -\\frac{1}{2}(x-2) \\).

الوحدة 2

القاعدة 2: الدائرة (Circles)

الدائرة هي مجموعة النقاط التي تبعد مسافة ثابتة (\\(r\\)) عن نقطة ثابتة (\\(h,k\\)).

الصيغة القياسية للدائرة:
\\( (x - h)^2 + (y - k)^2 = r^2 \\)
حيث \\((h,k)\\) هو المركز و \\(r\\) هو نصف القطر.

الصيغة العامة: \\( x^2 + y^2 + Dx + Ey + F = 0 \\)
يتم إكمال المربع لتحويلها للصيغة القياسية.
Module 2

Rule 2: Circles

A circle is the set of points at a fixed distance (\\(r\\)) from a fixed point (\\(h,k\\)).

Standard Circle Equation:
\\( (x - h)^2 + (y - k)^2 = r^2 \\)
where \\((h,k)\\) is the center and \\(r\\) is the radius.

General Form: \\( x^2 + y^2 + Dx + Ey + F = 0 \\)
Complete the square to convert to standard form.

📌 مثال تطبيقي (Application):

جد مركز ونصف قطر الدائرة: \\( x^2 + y^2 - 6x + 4y - 12 = 0 \\).

الحل بإكمال المربع:
1. \\( (x^2 - 6x) + (y^2 + 4y) = 12 \\).
2. \\( (x^2 - 6x + 9) + (y^2 + 4y + 4) = 12 + 9 + 4 \\).
3. \\( (x - 3)^2 + (y + 2)^2 = 25 \\).
النتيجة: المركز \\((3, -2)\\)، نصف القطر \\(5\\).

📌 Application Example:

Find center and radius of: \\( x^2 + y^2 - 6x + 4y - 12 = 0 \\).

Solution by Completing the Square:
1. Group terms: \\( (x^2 - 6x) + (y^2 + 4y) = 12 \\).
2. Complete: \\( (x-3)^2 + (y+2)^2 = 25 \\).
Result: Center \\((3, -2)\\), Radius \\(5\\).

الوحدة 3

القاعدة 3: القطوع المخروطية (Conic Sections)

القطوع المخروطية ثلاثة أنواع رئيسية في CSCA:

1. القطع المكافئ (Parabola): \\( y = ax^2 + bx + c \\) أو \\( x = ay^2 + by + c \\)
الرأس = \\((-\\frac{b}{2a}, f(-\\frac{b}{2a}))\\)، البؤرة = الرأس ± \\(\\frac{1}{4a}\\)

2. القطع الناقص (Ellipse): \\( \\frac{x^2}{a^2} + \\frac{y^2}{b^2} = 1 \\)
\\( c^2 = a^2 - b^2 \\) (للبؤرتين حيث \\(a\\) هو نصف المحور الأكبر)

3. القطع الزائد (Hyperbola): \\( \\frac{x^2}{a^2} - \\frac{y^2}{b^2} = 1 \\)
\\( c^2 = a^2 + b^2 \\)، الخطان التقاربان: \\( y = \\pm \\frac{b}{a}x \\)
Module 3

Rule 3: Conic Sections

Three main conic sections tested in CSCA:

1. Parabola: \\( y = ax^2 + bx + c \\) or \\( x = ay^2 + by + c \\)
Vertex = \\((-\\frac{b}{2a}, f(-\\frac{b}{2a}))\\), Focus = vertex ± \\(\\frac{1}{4a}\\)

2. Ellipse: \\( \\frac{x^2}{a^2} + \\frac{y^2}{b^2} = 1 \\)
\\( c^2 = a^2 - b^2 \\) (for foci, \\(a\\) is semi-major axis)

3. Hyperbola: \\( \\frac{x^2}{a^2} - \\frac{y^2}{b^2} = 1 \\)
\\( c^2 = a^2 + b^2 \\), Asymptotes: \\( y = \\pm \\frac{b}{a}x \\)

📌 مثال تطبيقي (Application):

لقطع ناقص معادلته: \\( \\frac{x^2}{25} + \\frac{y^2}{9} = 1 \\). أوجد إحداثيات البؤرتين.

الحل:
1. \\( a^2 = 25, b^2 = 9 \\).
2. \\( c^2 = a^2 - b^2 = 25 - 9 = 16 \\implies c = 4 \\).
3. المقام الأكبر تحت \\(x\\)، لذا المحور الأكبر أفقي.
النتيجة: \\( F_1(4, 0) \\) و \\( F_2(-4, 0) \\).

📌 Application Example:

Ellipse: \\( \\frac{x^2}{25} + \\frac{y^2}{9} = 1 \\). Find the foci coordinates.

Solution:
1. \\( a^2 = 25, b^2 = 9 \\).
2. \\( c^2 = 16 \\implies c = 4 \\).
3. Major axis is horizontal (larger denom under \\(x\\)).
Result: \\( F_1(4, 0) \\), \\( F_2(-4, 0) \\).

الوحدة 4

القاعدة 4: المتجهات (Vectors)

المتجه هو كمية لها مقدار واتجاه. نمثله كزوج مرتب \\( \\vec{v} = (v_1, v_2) \\).

العمليات الأساسية على المتجهات:
الطول (المقدار): \\( |\\vec{v}| = \\sqrt{v_1^2 + v_2^2} \\)
الجمع: \\( \\vec{a} + \\vec{b} = (a_1+b_1, a_2+b_2) \\)
الضرب الداخلي (Dot Product): \\( \\vec{a} \\cdot \\vec{b} = a_1b_1 + a_2b_2 \\)
التعامد: \\( \\vec{a} \\perp \\vec{b} \\iff \\vec{a} \\cdot \\vec{b} = 0 \\)
متجه الوحدة: \\( \\hat{u} = \\frac{\\vec{u}}{|\\vec{u}|} \\)
Module 4

Rule 4: Vectors

A vector has magnitude and direction. Represented as an ordered pair \\( \\vec{v} = (v_1, v_2) \\).

Basic Vector Operations:
Magnitude: \\( |\\vec{v}| = \\sqrt{v_1^2 + v_2^2} \\)
Addition: \\( \\vec{a} + \\vec{b} = (a_1+b_1, a_2+b_2) \\)
Dot Product: \\( \\vec{a} \\cdot \\vec{b} = a_1b_1 + a_2b_2 \\)
Orthogonality: \\( \\vec{a} \\perp \\vec{b} \\iff \\vec{a} \\cdot \\vec{b} = 0 \\)
Unit Vector: \\( \\hat{u} = \\frac{\\vec{u}}{|\\vec{u}|} \\)

📌 مثال تطبيقي (Application):

هل المتجهان \\( \\vec{u}(2, -3) \\) و \\( \\vec{v}(6, 4) \\) متعامدان؟

الحل:
1. الضرب الداخلي: \\( 2\\times6 + (-3)\\times4 = 12 - 12 = 0 \\).
2. بما أن الناتج صفر ⟹ المتجهان متعامدان.
النتيجة: نعم، متعامدان.

مثال آخر: طول \\(\\vec{v}(3,4)\\) = \\(\\sqrt{9+16}=5\\).

📌 Application Example:

Are vectors \\( \\vec{u}(2, -3) \\) and \\( \\vec{v}(6, 4) \\) perpendicular?

Solution:
1. Dot product: \\( 12 - 12 = 0 \\).
2. Zero result ⟹ vectors are orthogonal.
Result: Yes, they are perpendicular.

Another: Length of \\(\\vec{v}(3,4)\\) = \\(\\sqrt{9+16}=5\\).

الوحدة 5

القاعدة 5: الأعداد المركبة (Complex Numbers)

العدد المركب \\( z = a + bi \\) حيث \\( i = \\sqrt{-1} \\)، \\(a\\) هو الجزء الحقيقي، \\(b\\) هو الجزء التخيلي.

العمليات:
الجمع: \\( (a+bi) + (c+di) = (a+c) + (b+d)i \\)
الضرب: \\( (a+bi)(c+di) = (ac-bd) + (ad+bc)i \\)
المرافق: \\( \\bar{z} = a - bi \\)
المعيار: \\( |z| = \\sqrt{a^2 + b^2} \\)

الصيغة القطبية: \\( z = r(\\cos\\theta + i\\sin\\theta) \\)
مبرهنة ديموافر: \\( (\\cos\\theta + i\\sin\\theta)^n = \\cos(n\\theta) + i\\sin(n\\theta) \\)
Module 5

Rule 5: Complex Numbers

Complex number \\( z = a + bi \\) where \\( i = \\sqrt{-1} \\), \\(a\\) is real part, \\(b\\) is imaginary part.

Operations:
Addition: \\( (a+bi) + (c+di) = (a+c) + (b+d)i \\)
Multiplication: \\( (a+bi)(c+di) = (ac-bd) + (ad+bc)i \\)
Conjugate: \\( \\bar{z} = a - bi \\)
Modulus: \\( |z| = \\sqrt{a^2 + b^2} \\)

Polar Form: \\( z = r(\\cos\\theta + i\\sin\\theta) \\)
De Moivre's Theorem: \\( (\\cos\\theta + i\\sin\\theta)^n = \\cos(n\\theta) + i\\sin(n\\theta) \\)

📌 مثال تطبيقي (Application):

احسب \\( (1 + i)(2 - 3i) \\).

الحل:
1. نوزع: \\( 1\\cdot2 + 1\\cdot(-3i) + i\\cdot2 + i\\cdot(-3i) \\).
2. \\( = 2 - 3i + 2i - 3i^2 \\).
3. بما أن \\( i^2 = -1 \\): \\( = 2 - i - 3(-1) = 2 - i + 3 \\).
النتيجة: \\( 5 - i \\).

📌 Application Example:

Calculate \\( (1 + i)(2 - 3i) \\).

Solution:
1. Distribute: \\( 2 - 3i + 2i - 3i^2 \\).
2. Since \\( i^2 = -1 \\): \\( 2 - i + 3 \\).
Result: \\( 5 - i \\).

الوحدة 6

القاعدة 6: الهندسة الصلبة (Solid Geometry)

حساب المساحات السطحية والحجوم للمجسمات الأساسية.

قوانين هامة:
المكعب: الحجم \\( V = a^3 \\)، المساحة \\( S = 6a^2 \\)
متوازي المستطيلات: \\( V = abc \\)، \\( S = 2(ab+bc+ac) \\)
الكرة: \\( V = \\frac{4}{3}\\pi r^3 \\)، \\( S = 4\\pi r^2 \\)
الأسطوانة: \\( V = \\pi r^2 h \\)، \\( S = 2\\pi r^2 + 2\\pi rh \\)
المخروط: \\( V = \\frac{1}{3}\\pi r^2 h \\)، \\( S = \\pi r^2 + \\pi r l \\) (\\(l\\) = الارتفاع الجانبي)
الهرم: \\( V = \\frac{1}{3} \\times \\text{مساحة القاعدة} \\times h \\)
Module 6

Rule 6: Solid Geometry

Calculating surface areas and volumes of basic solids.

Key Formulas:
Cube: \\( V = a^3 \\), \\( S = 6a^2 \\)
Rectangular Prism: \\( V = abc \\), \\( S = 2(ab+bc+ac) \\)
Sphere: \\( V = \\frac{4}{3}\\pi r^3 \\), \\( S = 4\\pi r^2 \\)
Cylinder: \\( V = \\pi r^2 h \\), \\( S = 2\\pi r^2 + 2\\pi rh \\)
Cone: \\( V = \\frac{1}{3}\\pi r^2 h \\), \\( S = \\pi r^2 + \\pi r l \\) (\\(l\\) = slant height)
Pyramid: \\( V = \\frac{1}{3} \\times \\text{base area} \\times h \\)

📌 مثال تطبيقي (Application):

كرة نصف قطرها 6 سم. احسب حجمها ومساحتها السطحية.

الحل:
1. الحجم: \\( V = \\frac{4}{3}\\pi (6)^3 = \\frac{4}{3}\\pi \\times 216 = 288\\pi \\approx 904.78 \\) سم\\(^3\\).
2. المساحة: \\( S = 4\\pi (6)^2 = 144\\pi \\approx 452.39 \\) سم\\(^2\\).
النتيجة: الحجم \\( 288\\pi \\) سم\\(^3\\) والمساحة \\( 144\\pi \\) سم\\(^2\\).

📌 Application Example:

A sphere has radius 6 cm. Find its volume and surface area.

Solution:
1. Volume: \\( V = \\frac{4}{3}\\pi (216) = 288\\pi \\) cm\\(^3\\).
2. Surface Area: \\( S = 4\\pi (36) = 144\\pi \\) cm\\(^2\\).
Result: \\( V = 288\\pi \\) cm\\(^3\\), \\( S = 144\\pi \\) cm\\(^2\\).

🎯 نصيحة CSCA

  • حول معادلة الدائرة والقطوع للصيغة القياسية (\\(=1\\)) قبل البدء في الحساب مباشرة.
  • لا تخلط بين التوازي (نفس الميل) والتعامد (مقلوب عكسي الإشارة).
  • في الأعداد المركبة، تذكر \\(i^2 = -1\\)، \\(i^3 = -i\\)، \\(i^4 = 1\\) (دورة كل 4).
  • في المتجهات، إذا كان الضرب الداخلي صفراً ⟹ تعامد تام.
  • في القطوع المخروطية، افرق بين \\(c^2 = a^2 - b^2\\) (ناقص) للقطع الناقص و \\(c^2 = a^2 + b^2\\) (زائد) للقطع الزائد.
  • حفظ قوانين الحجوم والمساحات إلزامي — 2-3 أسئلة تأتي مباشرة منها.

🎯 CSCA Exam Tips

  • Convert circle and conic equations to standard form (\\(=1\\)) before calculating.
  • Don't confuse parallel (same slope) with perpendicular (negative reciprocal).
  • Complex numbers: remember \\(i^2 = -1\\), \\(i^3 = -i\\), \\(i^4 = 1\\) (cycle of 4).
  • Vectors: dot product = 0 ⟹ perfect orthogonality.
  • Conics: distinguish \\(c^2 = a^2 - b^2\\) (ellipse) from \\(c^2 = a^2 + b^2\\) (hyperbola).
  • Memorize volume/surface formulas — 2-3 direct questions come from them.

الوحدة 1 (تكميلي)

القاعدة الإضافية: المسافة ونقطة المنتصف

مسافة بين نقطتين: \\( d = \\sqrt{(x_2-x_1)^2 + (y_2-y_1)^2} \\)

نقطة المنتصف: \\( M = \\left(\\frac{x_1+x_2}{2}, \\frac{y_1+y_2}{2}\ ight) \\)

المسافة من نقطة لمستقيم: \\( d = \\frac{|Ax_0+By_0+C|}{\\sqrt{A^2+B^2}} \\)
حيث المستقيم بصيغة \\(Ax+By+C=0\\) والنقطة \\((x_0,y_0)\\).
Module 1 (Extra)

Extra Rule: Distance & Midpoint

Distance between two points: \\( d = \\sqrt{(x_2-x_1)^2 + (y_2-y_1)^2} \\)

Midpoint: \\( M = \\left(\\frac{x_1+x_2}{2}, \\frac{y_1+y_2}{2}\ ight) \\)

Distance from point to line: \\( d = \\frac{|Ax_0+By_0+C|}{\\sqrt{A^2+B^2}} \\)
where line is \\(Ax+By+C=0\\) and point is \\((x_0,y_0)\\).

📌 مثال تطبيقي (Application):

جد المسافة بين النقطتين \\((1, 2)\\) و \\((4, 6)\\) ثم نقطة المنتصف بينهما.

الحل:
1. المسافة: \\( d = \\sqrt{(4-1)^2 + (6-2)^2} = \\sqrt{9+16} = \\sqrt{25} = 5 \\).
2. نقطة المنتصف: \\( M = (\\frac{1+4}{2}, \\frac{2+6}{2}) = (2.5, 4) \\).
النتيجة: المسافة = 5، المنتصف = \\((2.5, 4)\\).

📌 Application Example:

Find distance between \\((1, 2)\\) and \\((4, 6)\\) and midpoint.

Solution:
1. Distance: \\( \\sqrt{9+16} = 5 \\).
2. Midpoint: \\( (2.5, 4) \\).
Result: Distance = 5, Midpoint = \\((2.5, 4)\\).
